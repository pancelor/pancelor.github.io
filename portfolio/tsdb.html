engine

For this game, I wrote a custom javascript engine. It runs in the browser, using the html <canvas> tag to display the game. I've learned a whole lot about game architecture since starting this project, and I sporadically get the urge to completely rewrite this in, say, C... but for now it's written in javascript :)
