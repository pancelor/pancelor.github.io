Tetris Clone (may 2020)

(this page is under construction!)

I did this project one week when I neeeded a break from my bigger projects. I decided I wanted to get a visceral feel for how hard it is to finish a game; surely it's hard? But how hard? So I decided to clone Tetris (the gameboy version) down to a tiny level of detail to see how much work it would take.

Turns out, it takes a lot of work :)
