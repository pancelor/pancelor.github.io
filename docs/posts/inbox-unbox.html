Inbox Unbox (mar 2019 - ?)

This is my upcoming puzzle game about pushing boxes into other boxes. It's 10~30 hours long? (It's hard to estimate puzzle game times...) You can't play it yet, but following me is the best way to get updates about it at the moment. (I'm also working on putting together a mailing list, but that's not ready yet -- come back in a week!)

Inspired by an excellent small game called sokosoko.

engine

For this game, I wrote a custom javascript engine. It runs in the browser, using the html <canvas> tag to display the game. I've learned a whole lot about game architecture since starting this project, and I sporadically get the urge to completely rewrite this in, say, C... but for now it's written in javascript :)
